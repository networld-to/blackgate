Host your page, e.g. your blog, in a matter of minutes
without ever bothering about the hosting part.

The Distributed Hosting Engine is a work in progress project that takes away from you the hassle to figure out how to host your page. It proposes a new type of blockchain with related protocol and reference implementation, that allows you host your page in a self-managed, distributed network.

Our team works currently on the specification and on a reference implementation. For that we need any help that we can get. If you want to get involved checkout the GitHub repository or contact us via blackgate@networld.to
